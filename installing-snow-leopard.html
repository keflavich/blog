I'm going to attempt to install snow leopard today. This post will serve as a record of the difficulties I run into. Things to install (and ensure they are 64-bit):

  • ipython
  • numpy
  • scipy
  • matplotlib
  • stsci-python
  • starlink
  • gfortran
  • latex
  • idl (check)
